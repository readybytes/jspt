Suppose you want, Profile Type1 users Can not send Messages to Profile Type2 users then perform as following:

  1. General Parameters
    • Resource Accessor -> Select Profile Type1.

  2. Rule Parameters
    • Resource Owner -> Select ProfileType2.
    • Maximum message Count -> Set Maximum messages Profile type1 user can send to profile type2 user
    • Applicable to Friend Also -> Set Yes only if you want this Rule to be Applicable on ProfileType2 Users Friend's Also
