Suppose you want, Profile Type1/Plan1 users Can not send Messages to Profile Type2/Plan2 users then perform as following:

  1. General Parameters
    • Resource Accessor -> Select Profile Type1/Plan1.

  2. Rule Parameters
    • Resource Owner -> Select Profile Type2/Plan2.
    • Maximum message Count -> Set Maximum messages Profile Type1/Plan1 user can send to Profile Type2/Plan2 user
    • Applicable to Friend Also -> Set Yes only if you want this Rule to be Applicable on Profile Type2/Plan2 Users Friend's also.
