Suppose you want, Profile Type1 users Can not view selective fields(say template) of profiletype2 then perform as following:

  1. General Parameters
    • Resource Accessor -> Select Profile Type1.

  2. Rule Parameters
    • Profiletype -> Select field you want to restrict(template) -> Select profile type2
    • Applicable to Friend Also -> Set Yes only if you want this Rule to be Applicable on ProfileType2 Users Friend's Also
