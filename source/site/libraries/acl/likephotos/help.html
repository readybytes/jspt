Suppose you want, Profile Type1/Plan1 users Can not Like Dislike Photos created by Profile Type2/Plan2 then perform as following:

  1. General Parameters
    • Resource Accessor -> Select Profile Type1/Plan1.

  2. Rule Parameters
    • Resource Owner -> Select Profile Type2/Plan2.
    • Applicable to Friend Also -> Set Yes only if you want this Rule to be Applicable on Profile Type2/Plan2 Users Friend's also.
