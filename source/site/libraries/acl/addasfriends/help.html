Suppose you want, Profile Type1/Plan1 users cannot add as friend Profile Type2/Plan2 users then perform as following:

  1. General Parameters
    • Resource Accessor -> Select Profile Type1/Plan1.

  2. Rule Parameters
    • Resource Owner -> Select Profile Type2/Plan2.
    • Total Friends -> Set Maximum number of friends Profile Type1/Plan1 users can add as friend to Profile Type2/Plan2 users.
